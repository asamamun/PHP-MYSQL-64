1. General Language Features of PHP

PHP is a server-side scripting language with the following features:

  • Open-source
  • Interpreted language
  • Dynamic typing
  • Cross-platform
  • Embedded in HTML
  • Built-in database support
  • Object-Oriented Programming (OOP)

2. Difference Between include and require

include gives a warning if the file is not found and continues execution, while require results in a fatal error and stops execution if the file is not found.

3. Super Global Variables

Super global variables in PHP are accessible from any scope:

  • $_GET
  • $_POST
  • $_REQUEST
  • $_SESSION
  • $_COOKIE
  • $_SERVER
  • $_FILES
  • $_ENV

4. What Is an Array? How to Output an Array?

An array is a data structure that holds multiple values. You can output an array using print_r() or var_dump().

print_r($array);

5. Recursive Function

A recursive function calls itself to solve problems that can be broken into smaller sub-problems.

function factorial($n) { if ($n == 0) return 1; return $n * factorial($n - 1); }

6. Passing Argument by Reference

Passing by reference allows a function to modify the original variable.

function increment(&$value) { $value++; }

7. PHP and Apache Configuration Files

PHP configuration: php.ini

Apache configuration: httpd.conf, .htaccess

8. Constant Declaration

Constants are values that cannot be changed once declared. Use define() to declare constants.

define("PI", 3.14159);

9. Type Casting and Type Juggling

Type casting is explicit conversion between data types. Type juggling is PHP's automatic conversion of data types.

$var = (int) $value;

10. Ternary Operator and Null Coalescing Operator

Use the ternary operator for simple conditionals and null coalescing for checking null values:

$status = ($age >= 18) ? "Adult" : "Minor"; $username = $_GET['user'] ?? 'Guest';

11. BREAK and CONTINUE Inside a Loop

BREAK exits the loop, and CONTINUE skips to the next iteration.

if ($i == 3) { continue; } if ($i == 5) { break; }
