PHP Class Inspection Functions

Function Description Syntax Example
class_exists() Checks if a class has been defined class_exists(string $class, bool $autoload = true): bool class_exists('MyClass') // true/false
get_class() Returns the class name of an object get_class(object $object): string get_class($obj) // 'MyClass'
get_declared_classes() Returns an array of all declared classes get_declared_classes(): array print_r(get_declared_classes())
is_subclass_of() Checks if an object/class is a subclass of another is_subclass_of(mixed $object, string $class): bool is_subclass_of($obj, 'ParentClass')
method_exists() Checks if a class method exists method_exists(object|string $object, string $method): bool method_exists($obj, 'myMethod')
property_exists() Checks if an object/class has a property property_exists(object|string $object, string $property): bool property_exists($obj, 'myProperty')
get_class_methods() Gets all methods of a class get_class_methods(object|string $object): array print_r(get_class_methods('MyClass'))
get_class_vars() Gets default properties of a class get_class_vars(string $class): array print_r(get_class_vars('MyClass'))
get_object_vars() Gets properties of an object get_object_vars(object $object): array print_r(get_object_vars($obj))
get_parent_class() Retrieves the parent class name get_parent_class(object|string $object): string|false get_parent_class($obj) // 'ParentClass'
interface_exists() Checks if an interface exists interface_exists(string $interface, bool $autoload = true): bool interface_exists('MyInterface')
trait_exists() Checks if a trait exists trait_exists(string $trait, bool $autoload = true): bool trait_exists('MyTrait')
is_a() The is_a() function returns TRUE if object belongs to a class of type class_name or if it belongs to a class that is a child of class_name. boolean is_a(object object, string class_name) is_a($obj, 'MyClass')
Usage Notes:
  • These functions are useful for reflection and introspection
  • Many accept either class names (strings) or object instances
  • The $autoload parameter determines if the autoloader should be called
  • For more advanced reflection, see PHP's ReflectionClass
