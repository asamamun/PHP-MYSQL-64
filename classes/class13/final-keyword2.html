PHP final Keyword

The final keyword prevents child classes from overriding methods or constants, and prevents class inheritance entirely.

1. Final Methods

class ParentClass {
    // This method cannot be overridden
    final public function criticalOperation() {
        return "This operation must not be changed";
    }
    
    public function normalMethod() {
        return "This can be overridden";
    }
}

class ChildClass extends ParentClass {
    public function normalMethod() {
        return "Overridden version";
    }
    
    // This would cause a fatal error:
    /*
    final public function criticalOperation() {
        return "Trying to override";
    }
    */
}
Note: Attempting to override a final method results in a fatal error:
"Cannot override final method ParentClass::criticalOperation()"

2. Final Classes

final class SecureDatabase {
    public function connect() {
        return "Secure connection established";
    }
}

// This would cause a fatal error:
/*
class HackAttempt extends SecureDatabase {
    public function connect() {
        return "Hacked!";
    }
}
*/

// But can be used normally:
$db = new SecureDatabase();
echo $db->connect(); // Works fine
