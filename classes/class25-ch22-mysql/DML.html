DDL (Data Definition Language)

Defines or modifies database structure.

  • CREATE – Create tables/databases
  • ALTER – Change table structure
  • DROP – Delete tables/databases
  • TRUNCATE – Clear all data in a table
CREATE TABLE users (
  id INT PRIMARY KEY,
  name VARCHAR(50)
);

ALTER TABLE users ADD email VARCHAR(100);

DROP TABLE users;

TRUNCATE TABLE users;

DML (Data Manipulation) – CRUD

Manages the data inside tables.

  • Create: INSERT
  • Read: SELECT
  • Update: UPDATE
  • Delete: DELETE
INSERT INTO users (id, name)
VALUES (1, 'Alice');

SELECT * FROM users;

UPDATE users
SET name = 'Bob'
WHERE id = 1;

DELETE FROM users
WHERE id = 1;
