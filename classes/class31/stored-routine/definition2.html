Declaring and Setting Variables in MySQL

Syntax:

DECLARE variable_name datatype [DEFAULT value];
SET variable_name = value;

Example:

DECLARE total INT DEFAULT 0;
SET total = 5;

The BEGIN and END Block in MySQL Stored Procedures

Syntax:

BEGIN
   -- statements
END;

Example:

BEGIN
   DECLARE total INT DEFAULT 0;
   SET total = 10;
END;

IF-ELSEIF-ELSE in MySQL

Syntax:

IF condition THEN
   -- statements
ELSEIF another_condition THEN
   -- statements
ELSE
   -- statements
END IF;

Example:

IF score >= 90 THEN
   SET grade = 'A';
ELSEIF score >= 80 THEN
   SET grade = 'B';
ELSE
   SET grade = 'C';
END IF;

Iteration in MySQL

Types: LOOP, WHILE, REPEAT

Example using WHILE:

DECLARE i INT DEFAULT 1;
WHILE i <= 5 DO
   SET i = i + 1;
END WHILE;

LEAVE in MySQL

Syntax:

LEAVE label_name;

Example:

my_loop: LOOP
   IF counter > 10 THEN
      LEAVE my_loop;
   END IF;
   SET counter = counter + 1;
END LOOP;

LOOP in MySQL

Syntax:

label: LOOP
   -- statements
   LEAVE label;
END LOOP label;

Example:

my_loop: LOOP
   SET counter = counter + 1;
   IF counter >= 5 THEN
      LEAVE my_loop;
   END IF;
END LOOP my_loop;
