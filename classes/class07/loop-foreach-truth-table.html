Example 1: Basic foreach Loop


$colors = ["red", "green", "blue"];

foreach ($colors as $color) {
    echo $color . "<br>";
}

Execution Table:

Iteration $key (implicit) $color (value) Output
1 0 red red<br>
2 1 green green<br>
3 2 blue blue<br>

Example 2: foreach with Key => Value


$colors = [
    "primary" => "red",
    "secondary" => "green",
    "tertiary" => "blue"
];

foreach ($colors as $key => $color) {
    echo "$key: $color<br>";
}

Execution Table:

Iteration $key $color Output
1 primary red primary: red<br>
2 secondary green secondary: green<br>
3 tertiary blue tertiary: blue<br>
