Task 4: Bank Account with Transaction History

Implement a secure bank account class that tracks all transactions.

Requirements:

  • Class Name: BankAccount
  • Private Properties:
    • accountNumber (string)
    • balance (float)
    • transactions (array to store history)
  • Methods:
    • deposit($amount, $description)
      • Adds to balance
      • Records transaction with: amount, type ("deposit"), description, timestamp
    • withdraw($amount, $description)
      • Deducts from balance if funds suffice
      • Records transaction with: amount, type ("withdrawal"), description, timestamp
    • getBalance() - Returns current balance
    • getTransactionHistory() - Returns all transactions
    • getLastTransaction() - Returns most recent transaction

Test Case:

  1. Create account with initial balance $1000
  2. Deposit $500 ("Paycheck")
  3. Withdraw $200 ("Rent payment")
  4. Print transaction history (should show 3 entries including initial balance)
  5. Print last transaction (should show rent payment)
