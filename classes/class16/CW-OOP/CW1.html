OOP Classwork Tasks (PHP)

Task 1: TV Remote Control

Implement a TV remote control class with basic functionality.

Requirements:

  • Class Name: TVRemote
  • Properties:
    • brand (string)
    • batteryLevel (int, range 0-100)
    • isOn (boolean)
  • Methods:
    • turnOn() - Sets TV state to on
    • turnOff() - Sets TV state to off
    • changeBattery() - Resets battery to 100%
    • checkBattery() - Returns current battery level
    • pressButton($buttonName) - Simulates button press

Test Case:

Create a remote, turn on TV, press some buttons, check battery, then turn off.

Class Object Methods

Task 2: Bike Class

Create a class to represent a bicycle with gear and speed control.

Requirements:

  • Class Name: Bike
  • Properties:
    • brand (string)
    • gear (int, range 1-6)
    • speed (float, in km/h)
  • Methods:
    • changeGear($newGear) - Validates and changes gear
    • accelerate($increase) - Increases speed
    • brake($decrease) - Decreases speed (minimum 0)
    • displayStatus() - Shows current bike stats

Test Case:

Create a bike, accelerate, change gears, brake, and display status.

Encapsulation Validation

Task 3: Car with Engine

Implement a car class that contains an engine object (composition).

Requirements:

  • Classes:
    • Engine (with type and isRunning)
    • Car (contains an Engine object)
  • Engine Methods:
    • start()
    • stop()
  • Car Methods:
    • startCar()
    • stopCar()

Test Case:

Create a car with a V6 engine, start it, then stop it.

Composition Object Collaboration

Task 4: Bank Account

Create a secure bank account class with encapsulation.

Requirements:

  • Class Name: BankAccount
  • Private Properties:
    • accountNumber
    • balance
  • Methods:
    • deposit($amount) - Validates positive amounts
    • withdraw($amount) - Checks sufficient funds
    • getBalance()
    • displayAccountInfo()

Test Case:

Create account, deposit $100, withdraw $30, display info.

Task 5: Student Grading System

Implement inheritance with Student and GraduateStudent classes.

Requirements:

  • Parent Class: Student
    • Properties: name, studentId
    • Method: displayInfo()
  • Child Class: GraduateStudent
    • Additional Property: thesisTopic
    • Override displayInfo() to include thesis

Test Case:

Create a graduate student and call their displayInfo() method.
