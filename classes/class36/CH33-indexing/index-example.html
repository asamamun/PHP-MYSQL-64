Database indexing and searching

An index is an ordered (or indexed) subset of table columns, with each row entry pointing to its corresponding table row. Generally speaking, introducing indexing into your MySQL database development strategy gives you three advantages:

  • Query optimization
  • Uniqueness
  • Text searching: full-text indexing

Example of full text index


SELECT * FROM products WHERE MATCH(description) AGAINST ('phone');
SELECT * FROM products WHERE MATCH(description) AGAINST ('processor');
SELECT * FROM products WHERE MATCH(name,description) AGAINST ('processor');

SELECT * FROM products WHERE description LIKE '%processor%';
