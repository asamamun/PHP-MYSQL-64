Regular Expression Topics

1. Regular Expression

A regular expression (regex) is a sequence of characters that define a search pattern, mainly for use in pattern matching with strings.

Example:
Pattern: /cat/
Matches: "The cat sat on the mat."

2. Regular Expression Syntax (Perl)

Perl-style regular expressions are common in many programming languages. They support grouping, quantifiers, character classes, etc.

Example:
Pattern: /\d{3}-\d{2}-\d{4}/
Matches: "123-45-6789"

3. Regular Expression Modifiers

Modifiers change the way regex behaves, like making it case-insensitive or global.

Example:
Pattern: /hello/i
Matches: "Hello", "HELLO", "heLLo"

4. Regular Expression Metacharacters

Metacharacters are characters with special meaning in regex like ., *, +, ?.

Example:
Pattern: /a.c/
Matches: "abc", "a7c", "a_c"
